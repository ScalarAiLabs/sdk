## 🚀 Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/scalar-ai.git
cd scalar-ai
```

### 2. Install Dependencies

```bash
npm install
```

### 3. Setup Environment

```bash
cp .env.example .env
```

### 4. Run the Project

```bash
npm run dev
```
